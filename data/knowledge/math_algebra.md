# คณิตศาสตร์ — พีชคณิต (Algebra)

## นิพจน์พีชคณิต (Algebraic Expressions)

นิพจน์พีชคณิตประกอบด้วยตัวแปร ตัวเลข และเครื่องหมายดำเนินการ
เช่น `3x + 2`, `a² + b²`, `(x + y)(x - y)`

### พจน์ (Terms)
- **เอกพจน์ (Monomial)**: พจน์เดียว เช่น `5x`, `-3y²`
- **ทวิพจน์ (Binomial)**: สองพจน์ เช่น `x + 3`, `2a - b`
- **ไตรพจน์ (Trinomial)**: สามพจน์ เช่น `x² + 2x + 1`
- **พหุนาม (Polynomial)**: หลายพจน์

### การดำเนินการพื้นฐาน
- **การบวก/ลบ**: รวมพจน์ที่คล้ายกัน (like terms) เช่น `3x + 5x = 8x`
- **การคูณ**: ใช้กฎกระจาย (Distributive Law) `a(b + c) = ab + ac`
- **การหาร**: ตรวจสอบว่าตัวหารไม่เป็นศูนย์

## สมการ (Equations)

สมการคือข้อความที่แสดงว่าสองนิพจน์มีค่าเท่ากัน โดยมีเครื่องหมาย `=`

### สมการเชิงเส้น (Linear Equations)
รูปทั่วไป `ax + b = 0` (x เป็นตัวแปร, a ≠ 0)

การแก้สมการ:
1. ย้ายพจน์ที่มีตัวแปรไปไว้ข้างเดียวกัน
2. ย้ายค่าคงที่ไปอีกข้างหนึ่ง
3. หารด้วยสัมประสิทธิ์ของตัวแปร

ตัวอย่าง: `2x + 3 = 7` → `2x = 4` → `x = 2`

### สมการกำลังสอง (Quadratic Equations)
รูปทั่วไป `ax² + bx + c = 0` (a ≠ 0)

การแก้สมการ:
1. **แยกตัวประกอบ (Factoring)** เช่น `x² + 5x + 6 = (x + 2)(x + 3) = 0`
2. **สูตรกำลังสอง (Quadratic Formula)**: `x = (-b ± √(b² - 4ac)) / 2a`
3. **ทำให้เป็นกำลังสองสมบูรณ์ (Completing the Square)**

### ค่า discriminant (Δ)
Δ = b² - 4ac บอกจำนวนคำตอบ:
- Δ > 0: 2 คำตอบจริง
- Δ = 0: 1 คำตอบจริง (ซ้ำ)
- Δ < 0: ไม่มีคำตอบจริง (เป็นจำนวนเชิงซ้อน)

## ระบบสมการ (Systems of Equations)

### สมการเชิงเส้นสองตัวแปร
มีสองสมการ สองตัวแปร (x, y)

วิธีแก้:
1. **การแทนค่า (Substitution)**: แก้สมการหนึ่ง แล้วแทนค่าอีกสมการ
2. **การกำจัด (Elimination)**: บวก/ลบสมการเพื่อกำจัดตัวแปร
3. **การใช้เมทริกซ์**: ใช้ determinant หรือ inverse matrix

ตัวอย่าง:
```
x + y = 10
x - y = 2
```
แก้โดยการบวกสองสมการ: `2x = 12` → `x = 6` → `y = 4`

## อสมการ (Inequalities)

ใช้เครื่องหมาย `<`, `>`, `≤`, `≥`

กฎสำคัญ: เมื่อคูณหรือหารด้วยจำนวนลบ ต้องกลับเครื่องหมายอสมการ

ตัวอย่าง: `-2x > 6` → `x < -3`

## เลขยกกำลัง (Exponents)

### กฎพื้นฐาน
- `a^m × a^n = a^(m+n)`
- `a^m / a^n = a^(m-n)`
- `(a^m)^n = a^(mn)`
- `(ab)^n = a^n × b^n`
- `(a/b)^n = a^n / b^n`
- `a^0 = 1` (a ≠ 0)
- `a^(-n) = 1 / a^n`
- `a^(1/n) = ⁿ√a`

### กรณฑ์ (Radicals)
- `√a` คือรากที่สองของ a (a ≥ 0)
- `³√a` คือรากที่สามของ a
- `√(ab) = √a × √b`
- `√(a/b) = √a / √b`

## ลอการิทึม (Logarithms)

นิยาม: `log_b(x) = y` หมายถึง `b^y = x` (b > 0, b ≠ 1, x > 0)

### กฎพื้นฐาน
- `log_b(xy) = log_b(x) + log_b(y)`
- `log_b(x/y) = log_b(x) - log_b(y)`
- `log_b(x^n) = n × log_b(x)`
- `log_b(1) = 0`
- `log_b(b) = 1`
- `log_b(x) = log_c(x) / log_c(b)` (การเปลี่ยนฐาน)

### ลอการิทึมธรรมชาติ
- `ln(x)` คือ `log_e(x)` โดย e ≈ 2.71828
- `ln(e) = 1`
- `ln(1) = 0`
