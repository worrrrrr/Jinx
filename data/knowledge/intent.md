# Jinx Intent Knowledge Base

| Intent | Keywords (Thai) | Keywords (English) | Description |
| :--- | :--- | :--- | :--- |
| **solve** | คำนวณ, แก้, บวก, ลบ | solve, calculate | ใช้สำหรับงานคณิตศาสตร์ |
| **edit** | แก้ไข, ปรับปรุง, เปลี่ยน | edit, update, modify | ใช้สำหรับแก้ไขไฟล์หรือข้อมูล |
| **search** | ค้นหา, หา, สืบค้น | search, find, query | ใช้สำหรับค้นหาข้อมูล |
| **analyze** | วิเคราะห์, สรุป, ตรวจสอบ | analyze, summarize, examine | ใช้สำหรับงานวิเคราะห์เชิงลึก |
| **create** | สร้าง, ทำ, เขียน | create, new, generate | ใช้สำหรับสร้างไฟล์หรือโค้ดใหม่ |
| **execute** | รัน, เริ่มทำงาน | run, execute | ใช้สำหรับรันไฟล์สคริปต์ |
| **translate** | แปล, แปลภาษา, แปลความ | translate, language translation | ใช้สำหรับแปลข้อความระหว่างภาษา |
| **explain** | อธิบาย, ชี้แจง, บอกกล่าว | explain, describe, clarify | ใช้สำหรับอธิบายแนวคิดหรือขั้นตอน |
| **compare** | เปรียบเทียบ, เทียบ, ต่างกัน | compare, contrast, difference | ใช้สำหรับเปรียบเทียบสิ่งของหรือข้อมูล |
| **check_grammar** | ตรวจสอบไวยากรณ์, เช็กภาษา | check grammar, grammar check, proofread | ใช้สำหรับตรวจสอบความถูกต้องทางภาษา |
| **suggest** | แนะนำ, เสนอ, ให้คำแนะนำ | suggest, recommend, advise | ใช้สำหรับให้คำแนะนำหรือข้อเสนอ |
| **format** | จัดรูปแบบ, จัดระเบียบ | format, beautify, prettify | ใช้สำหรับปรับแต่งรูปแบบโค้ดหรือข้อความ |
| **convert** | แปลง, เปลี่ยนหน่วย, แปลงหน่วย | convert, transform, unit conversion | ใช้สำหรับแปลงหน่วยหรือรูปแบบข้อมูล |
| **ask** | ถาม, สอบถาม, ถามคำถาม | ask, query, question | ใช้สำหรับตั้งคำถามหรือขอข้อมูลเพิ่ม |
| **example** | ยกตัวอย่าง, ตัวอย่าง, ตัวอย่างเช่น | example, demonstrate, illustrate | ใช้สำหรับยกตัวอย่างประกอบการอธิบาย |
| **summarize** | สรุป, สรุปความ, ย่อ | summarize, sum up, abstract | ใช้สำหรับย่อข้อความหรือดึงใจความสำคัญ |
| **paraphrase** | ถอดความ, เรียบเรียงใหม่ | paraphrase, rephrase, reword | ใช้สำหรับเขียนข้อความใหม่โดยคงความหมายเดิม |
| **debug** | แก้บั๊ก, ดีบั๊ก, ตรวจสอบบั๊ก | debug, fix bug, troubleshoot | ใช้สำหรับค้นหาและแก้ไขข้อผิดพลาดของโปรแกรม |
| **optimize** | เพิ่มประสิทธิภาพ, ปรับแต่ง | optimize, improve performance, tune | ใช้สำหรับทำให้โค้ดหรือระบบทำงานดีขึ้น |
| **test** | ทดสอบ, ตรวจสอบการทำงาน | test, run test, test case | ใช้สำหรับทดสอบฟังก์ชันหรือสคริปต์ |
| **deploy** | ใช้งาน, นำไปใช้, อัปโหลด | deploy, release, publish | ใช้สำหรับนำโค้ดหรือแอปขึ้นระบบจริง |
| **backup** | สำรองข้อมูล, แบ็กอัป | backup, back up, copy | ใช้สำหรับสำเนาข้อมูลเพื่อป้องกันการสูญเสีย |
| **import** | นำเข้า, อิมพอร์ต | import, load, read | ใช้สำหรับดึงข้อมูลจากภายนอกเข้ามา |
| **export** | ส่งออก, เอ็กซ์พอร์ต | export, save, write | ใช้สำหรับบันทึกข้อมูลออกไปภายนอก |
| **schedule** | กำหนดเวลา, จัดตาราง | schedule, set time, cron | ใช้สำหรับกำหนดเวลาทำงานอัตโนมัติ |
| **remind** | เตือน, เตือนความจำ | remind, alert, notification | ใช้สำหรับแจ้งเตือนหรือเตือนความจำ |
| **refactor** | ปรับโครงสร้าง, แก้โค้ดใหม่ | refactor, restructure, reorganize | ใช้สำหรับปรับปรุงโค้ดให้ดีขึ้นโดยไม่เปลี่ยนการทำงาน |
| **document** | เขียนเอกสาร, ทำคู่มือ | document, write doc, comment | ใช้สำหรับสร้างเอกสารประกอบโค้ดหรือระบบ |
| **validate** | ตรวจสอบความถูกต้อง, ตรวจสอบข้อมูล | validate, verify, check correctness | ใช้สำหรับยืนยันว่าข้อมูลหรือผลลัพธ์ถูกต้อง |
| **monitor** | ตรวจสอบสถานะ, ติดตาม | monitor, watch, observe | ใช้สำหรับดูสถานะหรือสุขภาพของระบบ |
| **visualize** | แสดงภาพ, พล็อตกราฟ | visualize, plot, chart | ใช้สำหรับสร้างกราฟหรือภาพจากข้อมูล |
| **train** | เทรน, ฝึกโมเดล | train, fit, learning | ใช้สำหรับสร้างโมเดลแมชชีนเลิร์นนิง |
| **predict** | ทำนาย, พยากรณ์ | predict, forecast, infer | ใช้สำหรับใช้โมเดลทำนายผลลัพธ์ |
| **clean** | ทำความสะอาดข้อมูล, จัดการข้อมูล | clean, tidy, preprocess | ใช้สำหรับล้างหรือปรับข้อมูลก่อนใช้งาน |
| **merge** | รวม, ผสาน, merge | merge, combine, join | ใช้สำหรับรวมไฟล์หรือข้อมูลหลายแหล่งเข้าเป็นหนึ่ง |
| **split** | แยก, ตัดแบ่ง | split, divide, separate | ใช้สำหรับแยกไฟล์หรือข้อมูลออกเป็นส่วนย่อย |
| **chat** | คุย, พูดคุย, สนทนา, เรื่องทั่วไป | chat, talk, casual conversation | ใช้สำหรับพูดคุยเรื่องทั่วไปที่ไม่ต้องการวิเคราะห์หรือแก้ปัญหาเฉพาะ |
| **greet** | สวัสดี, หวัดดี, ดีจ้า, ว่าไง | hello, hi, hey, good morning/afternoon | ใช้สำหรับทักทายและเปิดบทสนทนา |
| **farewell** | ลาก่อน, บ๊ายบาย, แล้วเจอกัน, ไปละ | goodbye, bye, see you, farewell | ใช้สำหรับบอกลาและจบการสนทนา |
| **appreciate** | ขอบคุณ, ขอบใจ, thanks, ขอบคุณมาก | thank you, thanks, appreciate, grateful | ใช้สำหรับแสดงความขอบคุณ |
| **empathize** | เสียใจด้วย, เห็นใจ, เข้าใจ, แย่จัง, เป็นห่วง | sorry to hear, feel for you, that's tough, I understand | ใช้สำหรับแสดงความเห็นอกเห็นใจเมื่อผู้ใช้เล่าเรื่องไม่ดีหรือรู้สึกแย่ |
| **support** | เป็นกำลังใจให้, สู้ๆ, เก่งมาก, ไม่เป็นไร | support, cheer up, you can do it, stay strong | ใช้สำหรับให้กำลังใจผู้ใช้ |
| **confirm** | ใช่, ถูกต้อง, จริงด้วย, ใช่เลย | yes, correct, that's right, confirm | ใช้สำหรับยืนยันความถูกต้องหรือตอบตกลง |
| **deny** | ไม่, ไม่ใช่, ผิด, ไม่เอา | no, incorrect, deny, not that | ใช้สำหรับปฏิเสธหรือบอกว่าไม่ถูกต้อง |
| **compliment** | ชม, เก่งจัง, เจ๋ง, ดีมาก | compliment, praise, well done, nice | ใช้สำหรับชมเชยผู้ใช้หรือสิ่งที่ผู้ใช้ทำ |
| **personalize** | เรียกชื่อ, เรียกอะไรดี, ปรับตามผู้ใช้, ฉันชื่อ... | call me, my name is, address me as, use my name | ใช้สำหรับให้บอทเรียกผู้ใช้ตามชื่อหรือคำที่ผู้ใช้ต้องการ (เช่น พี่, น้อง, ครู, หมอ) |
| **small_talk** | เป็นไงบ้าง, ทำอะไรอยู่, ว่างไหม | how are you, what's up, how's it going | ใช้สำหรับถามสารทุกข์สุขดิบหรือทักทายแบบไม่เป็นทางการ |
| **apologize** | ขอโทษ, โทษที, sorry, ขออภัย | sorry, apologize, my bad | ใช้สำหรับขอโทษเมื่อบอททำผิดหรือเข้าใจผิด |
