# คณิตศาสตร์ — สถิติและความน่าจะเป็น (Statistics & Probability)

## สถิติเบื้องต้น

### การเก็บรวบรวมข้อมูล
- **ข้อมูลดิบ (Raw Data)**: ข้อมูลที่ยังไม่ได้จัดระเบียบ
- **ประชากร (Population)**: กลุ่มทั้งหมดที่เราสนใจ
- **ตัวอย่าง (Sample)**: ส่วนหนึ่งของประชากรที่ใช้ในการศึกษา

### การวัดแนวโน้มเข้าสู่ส่วนกลาง (Measures of Central Tendency)
- **ค่าเฉลี่ย (Mean, μ หรือ x̄)**: ผลรวมของข้อมูลหารด้วยจำนวนข้อมูล
  - x̄ = (x₁ + x₂ + ... + xₙ) / n
- **มัธยฐาน (Median)**: ค่ากึ่งกลางเมื่อเรียงข้อมูลจากน้อยไปมาก
  - ถ้า n เป็นคี่: ตำแหน่ง (n+1)/2
  - ถ้า n เป็นคู่: ค่าเฉลี่ยของตำแหน่ง n/2 และ n/2+1
- **ฐานนิยม (Mode)**: ค่าที่ปรากฏบ่อยที่สุด

### การวัดการกระจาย (Measures of Dispersion)
- **พิสัย (Range)**: ค่าสูงสุด - ค่าต่ำสุด
- **ความแปรปรวน (Variance, σ²)**: ค่าเฉลี่ยของกำลังสองของส่วนเบี่ยงเบนจากค่าเฉลี่ย
  - ประชากร: σ² = Σ(xᵢ - μ)² / N
  - ตัวอย่าง: s² = Σ(xᵢ - x̄)² / (n-1)
- **ส่วนเบี่ยงเบนมาตรฐาน (Standard Deviation, σ หรือ s)**: √(variance)

### ควอร์ไทล์และเปอร์เซ็นไทล์
- **Q1 (ควอร์ไทล์ที่ 1)**: ค่ากึ่งกลางระหว่างค่าต่ำสุดกับมัธยฐาน
- **Q2**: มัธยฐาน
- **Q3 (ควอร์ไทล์ที่ 3)**: ค่ากึ่งกลางระหว่างมัธยฐานกับค่าสูงสุด
- **พิสัยระหว่างควอร์ไทล์ (IQR)**: Q3 - Q1

### การนำเสนอข้อมูล
- **ฮิสโตแกรม (Histogram)**: แท่งแสดงความถี่ของข้อมูล
- **แผนภาพกล่อง (Box Plot)**: แสดงค่าต่ำสุด Q1 Q2 Q3 สูงสุด
- **แผนภาพจุด (Scatter Plot)**: แสดงความสัมพันธ์ระหว่างสองตัวแปร

## ความน่าจะเป็น (Probability)

### นิยามพื้นฐาน
- **การทดลองสุ่ม (Random Experiment)**: การทดลองที่ผลลัพธ์ไม่แน่นอน
- **ผลลัพธ์ (Outcome)**: ผลที่เกิดขึ้นจากการทดลอง
- **แซมเปิลสเปซ (Sample Space, S)**: เซตของผลลัพธ์ทั้งหมด
- **เหตุการณ์ (Event, E)**: เซตย่อยของแซมเปิลสเปซ

### สูตรความน่าจะเป็น
P(E) = จำนวนผลลัพธ์ที่เกิดเหตุการณ์ E / จำนวนผลลัพธ์ทั้งหมด

- 0 ≤ P(E) ≤ 1
- P(S) = 1
- P(∅) = 0
- P(E') = 1 - P(E) (ความน่าจะเป็นของเหตุการณ์ตรงข้าม)

### กฎสำคัญ
- **ความน่าจะเป็นแบบเหตุการณ์เกิดร่วม (Intersection)**: P(A∩B)
- **ความน่าจะเป็นแบบเหตุการณ์รวม (Union)**: P(A∪B) = P(A) + P(B) - P(A∩B)
- **เหตุการณ์ที่เป็นอิสระต่อกัน (Independent Events)**: P(A∩B) = P(A) × P(B)
- **ความน่าจะเป็นแบบมีเงื่อนไข (Conditional Probability)**: P(A|B) = P(A∩B) / P(B)

### การนับ (Counting)
- **กฎการคูณ**: ถ้ามี m วิธีเลือกอย่างแรก และ n วิธีเลือกอย่างที่สอง รวม = m × n
- **การเรียงสับเปลี่ยน (Permutation)**: P(n,r) = n! / (n-r)!
- **การจัดหมู่ (Combination)**: C(n,r) = n! / (r!(n-r)!)

## การแจกแจงความน่าจะเป็น (Probability Distributions)

### การแจกแจงไม่ต่อเนื่อง (Discrete)
- **การแจกแจงทวินาม (Binomial)**: จำนวนความสำเร็จในการทดลองซ้ำ n ครั้ง
  - P(X=k) = C(n,k) × p^k × (1-p)^(n-k)
- **การแจกแจงปัวซอง (Poisson)**: จำนวนเหตุการณ์ในช่วงเวลาหนึ่ง
  - P(X=k) = (λᵏ × e^(-λ)) / k!

### การแจกแจงต่อเนื่อง (Continuous)
- **การแจกแจงปกติ (Normal Distribution)**: รูประฆังคว่ำ
  - N(μ, σ²): μ = ค่าเฉลี่ย, σ = ส่วนเบี่ยงเบนมาตรฐาน
  - การแจกแจงปกติมาตรฐาน Z: Z = (X - μ) / σ ~ N(0, 1)
- **กฎ 68-95-99.7**: 68% อยู่ใน μ±σ, 95% อยู่ใน μ±2σ, 99.7% อยู่ใน μ±3σ

## สหสัมพันธ์และการถดถอย (Correlation & Regression)

- **ค่าสหสัมพันธ์ (Correlation Coefficient, r)**: -1 ≤ r ≤ 1
  - r = 1: สัมพันธ์เชิงบวกสมบูรณ์
  - r = -1: สัมพันธ์เชิงลบสมบูรณ์
  - r = 0: ไม่มีความสัมพันธ์เชิงเส้น
- **การถดถอยเชิงเส้น (Linear Regression)**: y = mx + c (หาเส้นที่เหมาะสมที่สุด)
