## ⚔️ 2. โลหะ (Metal) — Ni (Introverted Intuition) + Ti (Introverted Thinking)

**ธาตุแห่งตรรกะ โครงสร้าง การวิเคราะห์**

| ระดับ | ลักษณะที่แสดงออก | Cognitive Function |
|-------|-----------------|-------------------|
| **ขาด (0-1)** | ไม่มีระบบในความคิด, ทำอะไรตามอารมณ์, ตัดสินใจลำบาก, ขาดการวางแผน | Ni ต่ำมาก, Ti อ่อน |
| **ต่ำ (1.5-2.5)** | คิดได้แต่ไม่ลึก, วางแผนได้ระดับหนึ่ง แต่ยังหลงลืมง่าย | Ni ปานกลาง, Ti พอใช้ |
| **สมดุล (3-4)** | คิดเป็นระบบ, มองภาพรวมเก่ง, แก้ปัญหาได้ดี, มีมาตรฐานชัดเจน | Ni สูง, Ti สูง — **INTJ/INFJ/ISTP** |
| **สูง (4.5-5.5)** | คิดมาก, วิเคราะห์ทุกอย่าง, เงียบขรึม, มีความเป็น perfectionist | Ni โดด, Ti แข็ง — มักเป็น **INTJ/INTP** |
| **เกิน (6+)** | Overthinking ขั้นรุนแรง, ชอบจับผิด, วิพากษ์วิจารณ์, เก็บตัวจัด, สุขภาพจิตเสียง่าย | Ni ควบคุมทุกอย่าง, Ti กลายเป็น rigid |

## 🔥 3. ไฟ (Fire) — Fi (Introverted Feeling)

**ธาตุแห่งพลังชีวิต ความรู้สึก แรงบันดาลใจ**

| ระดับ | ลักษณะที่แสดงออก | Cognitive Function |
|-------|-----------------|-------------------|
| **ขาด (0-1)** | ไม่รู้ว่าตัวเองรู้สึกอะไร, ขาดแพชชั่น, ทำตามคนอื่น, ดูจืดชืด | Fi ต่ำมาก — คล้าย **INTP/ISTP** |
| **ต่ำ (1.5-2.5)** | มีความรู้สึกแต่ไม่ค่อยแสดงออก, ต้องใช้เวลาถึงจะรู้ว่าตัวเองต้องการอะไร | Fi ปานกลาง |
| **สมดุล (3-4)** | มีแพชชั่น, ซื่อสัตย์ต่อความรู้สึก, อบอุ่น, มีเสน่ห์ | Fi ดี — **INFP/INFJ** |
| **สูง (4.5-5.5)** | อารมณ์แรง, ดราม่า, ยึดมั่นในคุณค่าตัวเอง, ต้องการการยอมรับ | Fi สูงมาก — **ENFP/INFP** |
| **เกิน (6+)** | เหวี่ยง, เอาแต่ใจ, โทษคนอื่น, ควบคุมอารมณ์ไม่ได้ | Fi toxic → กลายเป็น Fe ลบ |

## 🏔️ 4. ดิน (Earth) — Si (Introverted Sensing) + Te (Extroverted Thinking)

**ธาตุแห่งความมั่นคง ระบบ ประสิทธิภาพ**

| ระดับ | ลักษณะที่แสดงออก | Cognitive Function |
|-------|-----------------|-------------------|
| **ขาด (0-1)** | ไร้ระเบียบ, ไม่รับผิดชอบ, เปลี่ยนงานบ่อย, ใช้ชีวิตวันต่อวัน | Si ต่ำ, Te อ่อน — **ENFP/ENTP** |
| **ต่ำ (1.5-2.5)** | พอมีวินัยบ้าง แต่ยังขาดระบบ | Si/Te พอใช้ — **ISFP/INFP** |
| **สมดุล (3-4)** | จัดการชีวิตดี, ทำงานเป็นระบบ, ไว้ใจได้ | Si/Te ดี — **ISTJ/ESTJ/ENTJ** |
| **สูง (4.5-5.5)** | เจ้าระเบียบ, ชอบควบคุม, อนุรักษ์นิยม, ยึดติดกับกฎเกณฑ์ | Si/Te แข็ง — **ESTJ/ENTJ** |
| **เกิน (6+)** | หัวโบราณสุดขั้ว, บ้าอำนาจ, ไม่ยืดหยุ่น, กดขี่ผู้อื่น | Si/Te toxic → กลายเป็น bossy/controlling |

## 💧 5. น้ำ (Water) — Fe (Extroverted Feeling)

**ธาตุแห่งอารมณ์ การสื่อสาร ความเข้าอกเข้าใจ**

| ระดับ | ลักษณะที่แสดงออก | Cognitive Function |
|-------|-----------------|-------------------|
| **ขาด (0-1)** | ไม่เข้าใจความรู้สึกคนอื่น, สื่อสารห้วน ๆ, เหมือนมีกำแพง | Fe ต่ำ — **INTP/ISTP** |
| **ต่ำ (1.5-2.5)** | เข้าใจคนได้บ้าง แต่ไม่ถนัดแสดงออก | Fe ปานกลาง — **INFJ/INTJ** |
| **สมดุล (3-4)** | เห็นอกเห็นใจ, สื่อสารดี, ประนีประนอม, เข้ากับคนง่าย | Fe ดี — **ENFJ/ESFJ/INFJ** |
| **สูง (4.5-5.5)** | ต้องการทำให้ทุกคนพอใจ, หลีกเลี่ยงความขัดแย้ง, หมดพลังกับคนอื่น | Fe สูง — **ENFJ/ESFJ** |
| **เกิน (6+)** | จอมเสี้ยม, เปลี่ยนแปลงตัวเองไปตามกลุ่มจนเสียตัวตน, ถูกชักจูงง่าย | Fe toxic → manipulative |

## 🌿 6. ไม้ (Wood) — Se (Extroverted Sensing) + Ne (Extroverted Intuition)

**ธาตุแห่งการเติบโต ความยืดหยุ่น ความคิดสร้างสรรค์**

| ระดับ | ลักษณะที่แสดงออก | Cognitive Function |
|-------|-----------------|-------------------|
| **ขาด (0-1)** | ไม่ยืดหยุ่น, ติดกรอบ, ต้องวางแผนก่อนเสมอ, ไม่กล้าเสี่ยง | Se/Ne ต่ำ — **INTJ/INFJ/ISTJ** |
| **ต่ำ (1.5-2.5)** | เปิดรับสิ่งใหม่บ้าง แต่ยังระวังตัว | Se/Ne ปานกลาง — **ISFJ/ISTP** |
| **สมดุล (3-4)** | ปรับตัวเก่ง, มีความคิดริเริ่ม, ชอบเรียนรู้, หาโอกาสใหม่ ๆ | Se/Ne ดี — **ENTP/ENFP/ESFP** |
| **สูง (4.5-5.5)** | ทำอะไรตามสัญชาตญาณ, หลากหลายแต่ไม่ลึก, โลเล | Se/Ne สูง — **ESTP/ENTP** |
| **เกิน (6+)** | ฟุ้งซ่าน, เปลี่ยนทิศทางตลอด, ไม่มีวินัย, หมดแรงกับความใหม่ | Se/Ne toxic → scattered |

## 🧠 7. วิธีใช้ตารางนี้ใน `synthesize_mbti_enneagram`

```python
# หลังคำนวณ wood, fire, earth, metal, water
# ตัวอย่างการปรับ scholar (Ni/Ti) จากระดับโลหะ

if metal <= 1:           # ขาด
    scholar -= 2         # Ni/Ti ต่ำมาก
elif metal <= 2.5:       # ต่ำ
    scholar -= 1
elif metal <= 4:         # สมดุล
    scholar += 2         # ทำงานปกติ
elif metal <= 5.5:       # สูง
    scholar += 4         # Ni/Ti โดด (overthinking)
else:                    # เกิน
    scholar += 6         # Ni/Ti แข็งแบบ rigid
```
