# ชีววิทยาพื้นฐาน (Basic Biology)

## เซลล์ (Cells)

### ทฤษฎีเซลล์
1. สิ่งมีชีวิตทุกชนิดประกอบด้วยเซลล์
2. เซลล์คือหน่วยพื้นฐานของโครงสร้างและการทำงาน
3. เซลล์ทุกเซลล์เกิดจากเซลล์ที่มีอยู่ก่อน

### ชนิดของเซลล์
- **Prokaryote (โปรคาริโอต)**: ไม่มีนิวเคลียสและออร์แกเนลล์มีเยื่อหุ้ม (เช่น แบคทีเรีย)
- **Eukaryote (ยูคาริโอต)**: มีนิวเคลียสและออร์แกเนลล์มีเยื่อหุ้ม (เช่น สัตว์ พืช เชื้อรา)

### ออร์แกเนลล์สำคัญ
- **นิวเคลียส (Nucleus)**: บรรจุ DNA ควบคุมการทำงานของเซลล์
- **ไมโทคอนเดรีย (Mitochondria)**: แหล่งผลิตพลังงาน (ATP) — "โรงไฟฟ้าของเซลล์"
- **ไรโบโซม (Ribosome)**: สร้างโปรตีน
- **เอนโดพลาสมิกเรติคูลัม (ER)**: สร้างและขนส่งโปรตีน (Rough ER) และไขมัน (Smooth ER)
- **กอลจิแอพพาราตัส (Golgi Apparatus)**: จัดเรียงและส่งออกโปรตีน
- **ไลโซโซม (Lysosome)**: ย่อยสลายของเสียในเซลล์
- **คลอโรพลาสต์ (Chloroplast)**: ที่เกิดการสังเคราะห์ด้วยแสง (เฉพาะพืช)
- **แวคิวโอล (Vacuole)**: เก็บน้ำและสารต่างๆ (ในพืชมีขนาดใหญ่)

## พันธุศาสตร์ (Genetics)

### DNA และ RNA
- **DNA (Deoxyribonucleic Acid)**: สารพันธุกรรม เกลียวคู่ (double helix) ประกอบด้วยนิวคลีโอไทด์
  - นิวคลีโอไทด์: A (Adenine), T (Thymine), G (Guanine), C (Cytosine)
  - A จับคู่กับ T, G จับคู่กับ C
- **RNA (Ribonucleic Acid)**: เกลียวเดี่ยว ใช้ U (Uracil) แทน T

### Central Dogma
DNA → (Transcription) → RNA → (Translation) → Protein

- **Transcription**: DNA ถูกถอดรหัสเป็น mRNA
- **Translation**: mRNA ถูกแปลรหัสเป็นสายโปรตีนที่ไรโบโซม
- **Codon**: รหัส 3 นิวคลีโอไทด์บน mRNA แทนกรดอะมิโน 1 ตัว

### กฎของเมนเดล (Mendelian Genetics)
- **Gene (ยีน)**: หน่วยของพันธุกรรม
- **Allele (อัลลีล)**: รูปแบบของยีน (เด่น = Dominant, ด้อย = Recessive)
- **Homozygous**: มียีนเหมือนกันสองอัลลีล
- **Heterozygous**: มียีนต่างกันสองอัลลีล
- **Genotype**: องค์ประกอบทางพันธุกรรม
- **Phenotype**: ลักษณะที่แสดงออก

## ระบบในร่างกายมนุษย์

### ระบบย่อยอาหาร (Digestive System)
ปาก → หลอดอาหาร → กระเพาะอาหาร → ลำไส้เล็ก → ลำไส้ใหญ่ → ทวารหนัก
- ตับอ่อนสร้างเอนไซม์ ตับสร้างน้ำดี

### ระบบหายใจ (Respiratory System)
จมูก → หลอดลม → ปอด
- การแลกเปลี่ยนแก๊สเกิดขึ้นที่ถุงลม (Alveoli)
- O₂ เข้าสู่เลือด CO₂ ออกจากเลือด

### ระบบหมุนเวียนเลือด (Circulatory System)
- หัวใจมี 4 ห้อง: ซ้ายบน(LA), ขวาบน(RA), ซ้ายล่าง(LV), ขวาล่าง(RV)
- หลอดเลือดแดง (Artery): นำเลือดออกจากหัวใจ
- หลอดเลือดดำ (Vein): นำเลือดเข้าสู่หัวใจ
- เส้นเลือดฝอย (Capillary): แลกเปลี่ยนสาร

### ระบบประสาท (Nervous System)
- **สมอง (Brain)**: ควบคุมการทำงานของร่างกาย
- **ไขสันหลัง (Spinal Cord)**: เชื่อมต่อสมองกับร่างกาย
- **นิวรอน (Neuron)**: เซลล์ประสาท ส่งสัญญาณไฟฟ้าเคมี

## วิวัฒนาการ (Evolution)

### ทฤษฎีของดาร์วิน
1. สิ่งมีชีวิตมีการเปลี่ยนแปลงตลอดเวลา
2. การคัดเลือกโดยธรรมชาติ (Natural Selection): สิ่งมีชีวิตที่ปรับตัวได้ดีกว่าจะอยู่รอดและสืบพันธุ์
3. การอยู่รอดของสายพันธุ์ที่เหมาะสมที่สุด (Survival of the Fittest)

### หลักฐานวิวัฒนาการ
- ซากดึกดำบรรพ์ (Fossils)
- กายวิภาคเปรียบเทียบ (Comparative Anatomy)
- ความคล้ายคลึงของ DNA

## นิเวศวิทยา (Ecology)

### ระดับของระบบนิเวศ
- **สิ่งมีชีวิต (Organism)**: หน่วยเดียว
- **ประชากร (Population)**: สิ่งมีชีวิตชนิดเดียวกันในพื้นที่
- **ชุมชน (Community)**: หลายประชากรในพื้นที่เดียวกัน
- **ระบบนิเวศ (Ecosystem)**: ชุมชน + สิ่งแวดล้อม
- **ชีวมณฑล (Biosphere)**: ทุกระบบนิเวศบนโลก

### ห่วงโซ่อาหาร (Food Chain)
ผู้ผลิต (Producer) → ผู้บริโภคปฐมภูมิ → ผู้บริโภคทุติยภูมิ → ผู้บริโภคตติยภูมิ → ผู้ย่อยสลาย

### การถ่ายทอดพลังงาน
พลังงานจะลดลงประมาณ 90% ในแต่ละระดับของห่วงโซ่อาหาร (10% rule)
